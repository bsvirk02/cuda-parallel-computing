# CUDA Parallel Computing

High-performance parallel programming implementations using CUDA, solving computationally intensive problems on GPU hardware, built as an individual project at the University of Surrey.

## Tech Stack

- **Language:** CUDA C/C++
- **Platform:** NVIDIA GPU
- **Tools:** CUDA Toolkit

## Key Concepts Implemented

- Parallel thread organisation and execution on GPU
- Memory architecture optimisation and data locality
- Convolution and stencil operations
- Parallel histogram computation
- Parallel reduction algorithms
- Prefix-sum (scan) operations
- Performance optimisation through hyperparameter tuning

## About

Built as an individual university project exploring low-level GPU programming and parallel computing techniques using CUDA, demonstrating understanding of hardware-aware software optimisation.
